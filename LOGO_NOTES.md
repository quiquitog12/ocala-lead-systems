# Ocala Lead Systems — Logo Notes

## Files
- `logo-mark.svg` — square icon / app mark / social avatar
- `logo-horizontal.svg` — primary horizontal logo
- `favicon.svg` — simple favicon

## Direction
- Dark luxury base
- Gold premium accent to match the website
- Monogram feel for local-business trust + premium positioning
- Clean enough to work for web, favicon, and future business cards

## Recommended usage
- Website navbar: use `logo-horizontal.svg`
- Browser tab/favicon: use `favicon.svg`
- Instagram/Facebook profile: use `logo-mark.svg`
- Watermark / proposal docs: use `logo-mark.svg`
