## Recommended Workflow
1. Build target list by niche
2. Add businesses into CRM / Notion CSV
3. Personalize subject line + first sentence
4. Send Email 1
5. Log the action in the CRM
6. If no reply, follow cadence
7. If they reply, move status to `Responded`
8. If booked, move to `Booked Call`
9. If proposal sent, move to `Proposal Sent`
10. If closed, move to `Won`

## Suggested Statuses
